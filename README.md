# 🌱 ECO-fit — Landing Page

**ECO-fit** est une application mobile de fitness écoresponsable. Cette landing page a pour but de présenter l’app de manière engageante et interactive, tout en incitant les visiteurs à la télécharger dès sa sortie.

---

## ✨ Aperçu

🌍 *"Bouge pour toi. Agis pour la planète."*

Cette landing page propose :
- Un **loader animé** (spinner + message de chargement)
- Un **fond dynamique** avec des feuilles et emojis animés
- Une section **"Hero" immersive** avec slogan et call-to-action
- Des **cartes interactives** mettant en valeur les fonctionnalités clés
- Un **CTA final** clair et convaincant
- Un **design responsive** optimisé pour mobile

## 🛠️ Technologies utilisées

- **HTML5**
- **Tailwind CSS (CDN)**
- **JavaScript Vanilla**
- **Animations CSS personnalisées**
- Aucun framework externe requis ✅

---

## ▶️ Lancer en local

1. Clone ce dépôt :
```bash
git clone https://github.com/ton-profil/ecofit-landing.git
````

2. Ouvre `index.html` dans ton navigateur :

```bash
cd ecofit-landing
open index.html  # ou double-clique si sur Windows
```
